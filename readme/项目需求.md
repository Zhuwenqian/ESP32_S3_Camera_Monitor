ESP32-S3-CAM 监控项目需求
忽略库引用的错误
语言:C/C++(Arduino框架,主程序ino格式)
摄像头配置：
OV5640，分辨率720P，摄像头时钟22Mhz，其余配置见camera_pins.h
功能：
1.在web端显示监控视频（流式输出）
2.可以在web端控制监控摄像头的旋转角度（SG-90舵机）
3.持续分段（2分钟一段）录制视频，保存到ESP32-S3-CAM的SD卡中。当文件达到55GB时，自动删除最早的视频文件。
4.web端可以控制拍照，拍照后将照片保存到ESP32-S3-CAM的SD卡中。
